Задание 3.3. Изменить программу из п. 3.1 так, чтобы
она завершалась после третьего сигнала SIGINT.

Задание 3.1. Написать программу, бесконечно
выводящую в файл 1 раз в секунду значение внутреннего
счетчика (1 2 3 ...). Запустить ее в фоновом режиме
(myprogram &).

===

Программа выводит значение счетчика в файл file.txt.
С помощью команды signal(SIGINT, SIGIGT_listener) для сигнала SIGINT назначается функция SIGIGT_listener, 
выполняющая инкрементирование локальной статической переменной. При достижении определенного значения этой переменной
процесс завершается командой exit(EXIT_SUCCESS).

===

Запуск в фоновом режиме: ./task3_3_3 &
Пример работы на рисунке example.png. Процессу трижды отправляется сигнал SIGINT.
