Задание 1.2. Написать программу, порождающую дочерний процесс.
Затем и родительский, и дочерний процессы выводят значения
аргументов запуска
(int main (int argc, char *argv[])).
Добавить функцию, вызываемую при нормальном завершении работы
программы и проверить ее работу.

===

Программа выводит в двух процессах аргументы запуска, включая нулевой. 
При нормальном завершении программа выводит текст "Successful exit...".

===

Пример запуска: ./task3_1_2 arg1 25 0
